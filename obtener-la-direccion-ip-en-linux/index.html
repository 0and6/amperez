Obtener la dirección ip en linux

Para poder obtener la dirección ip solo basta con escribir el comando 

$ ip address

Como resultados nos va a mostrar algo similar a esto

1: lo: mtu 65536 qdisc noqueue state UNKNOWN group default qlen 1000
link/loopback 00:00:00:00:00:00 brd 00:00:00:00:00:00
inet 127.0.0.1/8 scope host lo
valid_lft forever preferred_lft forever
inet6 ::1/128 scope host
valid_lft forever preferred_lft forever
2: enp3s0f1: mtu 1500 qdisc pfifo_fast state DOWN group default qlen 1000
link/ether 54:ab:3a:aa:f7:a2 brd ff:ff:ff:ff:ff:ff
3: wlp2s0: mtu 1500 qdisc noqueue state UP group default qlen 1000
link/ether c8:ff:28:f1:3c:99 brd ff:ff:ff:ff:ff:ff
inet 192.168.100.61/24 brd 192.168.100.255 scope global dynamic noprefixroute wlp2s0
valid_lft 84579sec preferred_lft 84579sec
inet6 2806:261:492:59d:e25c:1a73:bbbb:bbbb/64 scope global dynamic noprefixroute
valid_lft 2591813sec preferred_lft 604613sec
inet6 fe80::e20:fa4:4909:85cb/64 scope link noprefixroute
valid_lft forever preferred_lft forever

De donde podemos obtener nuestra dirección ip para las distintas interfaces de red. En primer lugar tenemos la interfaz de loopback (lo), después la interfaz de ethernet (enp0s0f1) y por último la interfaz del wifi (wlp2s0).

La dirección ip viene marcada despues de “inet”; en el caso de ipv6 aparece como “inet6”.

Dirección ip loopback: inet 127.0.0.1/8

Dirección ip tarjeta ethernet: Sin dirección ip asignada

Dirección del wifi:

inet 192.168.100.61/24
inet6 2806:261:492:59d:e25c:1a73:bbbb:bbbb/64
